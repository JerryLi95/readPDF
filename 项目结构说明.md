# 项目目录结构说明

## 📁 目录组织

```
readPDF/
├── src/                          # 【源代码目录】开发文件
│   ├── extract_pdf_tables.py    # 主程序源代码
│   ├── build_exe.py             # 打包脚本
│   └── requirements.txt         # Python依赖包列表
│
├── release/                      # 【发布目录】可分发文件
│   ├── PDF表格提取工具.exe      # 可执行程序（打包后复制到这里）
│   ├── 使用说明.txt             # 用户使用说明
│   └── README.txt               # 发布包说明
│
├── docs/                         # 【文档目录】项目文档
│   ├── README.md                # 开发文档
│   └── 使用说明.txt             # 使用说明（副本）
│
├── build/                        # 【构建目录】PyInstaller临时文件（可删除）
├── dist/                         # 【输出目录】PyInstaller输出（可删除）
├── INPUT/                        # 【测试目录】测试用PDF文件（开发用）
│
├── README.md                     # 项目说明（本文件）
├── .gitignore                    # Git忽略文件
└── PDF表格提取工具.spec         # PyInstaller配置文件（可删除）
```

## 📋 目录说明

### src/ - 源代码目录
**用途**: 存放所有源代码和开发相关文件
- `extract_pdf_tables.py`: 主程序源代码
- `build_exe.py`: 打包脚本（用于生成exe）
- `requirements.txt`: Python依赖包列表

**操作**: 
- 修改源代码时，编辑此目录下的文件
- 打包前需要先安装依赖：`pip install -r requirements.txt`

### release/ - 发布目录
**用途**: 存放可分发给最终用户的文件
- `PDF表格提取工具.exe`: 打包后的可执行程序
- `使用说明.txt`: 用户使用说明
- `README.txt`: 发布包说明

**操作**:
- 打包完成后，exe文件会自动复制到此目录
- 分发时，将此目录下的所有文件打包发送给用户

### docs/ - 文档目录
**用途**: 存放项目文档
- `README.md`: 开发文档和技术说明
- `使用说明.txt`: 使用说明文档

### build/ 和 dist/ - 构建目录
**用途**: PyInstaller打包时自动生成的临时文件
- `build/`: 构建过程中的临时文件
- `dist/`: PyInstaller的输出目录

**操作**: 这些目录可以随时删除，不会影响源代码

### INPUT/ - 测试目录
**用途**: 存放测试用的PDF文件（开发时使用）

## 🔧 开发工作流

### 1. 开发阶段
```bash
cd src
pip install -r requirements.txt
python extract_pdf_tables.py
```

### 2. 打包阶段
```bash
cd src
python build_exe.py
```
打包完成后，exe文件会自动复制到 `release/` 目录

### 3. 分发阶段
从 `release/` 目录获取文件，打包分发给用户

## 📦 分发文件清单

**最小分发**（推荐）:
- `release/PDF表格提取工具.exe`
- `release/使用说明.txt`

**完整分发**:
- `release/` 目录下的所有文件

## 🗑️ 可删除的文件/目录

以下文件/目录可以安全删除，不影响开发：
- `build/` - 构建临时文件
- `dist/` - PyInstaller输出（打包后exe已复制到release）
- `PDF表格提取工具.spec` - PyInstaller配置文件
- `INPUT/` - 测试文件（如果不需要）
- `提取结果.xlsx` - 测试输出文件

## ⚠️ 注意事项

1. **不要删除** `src/` 目录下的任何文件
2. **不要删除** `release/` 目录（分发时需要）
3. 修改源代码后需要重新打包才能更新exe文件
4. 打包脚本会自动将exe复制到release目录
